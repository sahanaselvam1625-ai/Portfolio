Healthcare Monitoring

Overview

This project focuses on analyzing hospital patient data to predict and visualize readmission risks. It provides an interactive Power BI dashboard that helps healthcare providers monitor patient demographics, insurance details, readmission trends, and health risk levels. The system supports data-driven decision-making to improve patient care quality and reduce unnecessary readmissions.

Features

Executive Summary Dashboard

>> Displays overall readmission rates and insurance provider analysis.
>> Highlights monthly readmission trends and admission types by insurance.

Readmission Risk Prediction Dashboard

>> Calculates and visualizes patient readmission risk scores.
>> Includes condition-wise risk distribution (e.g., diabetes, hypertension).
>> Provides risk band filters (High, Medium, Low) for quick analysis.

Patient Demographic Overview Dashboard

>> Shows total number of patients and their admission types.
>> Visualizes patients by age group, gender, and blood type.
>> Contains a detailed patient information table for easy reference.

Interactive Visuals

>> Real-time filtering for insurance types, risk levels, and demographics.
>> Charts, pie graphs, and treemaps for deeper data insights.

Decision Support

>> Helps hospitals identify high-risk patients early.
>> Assists in optimizing resources and improving treatment strategies.
